## What is SellDrop?

SellDrop is a modern e-commerce platform purpose-built for selling digital goods like files, serial keys, services, and more. Launch a storefront, accept payments worldwide, and manage your entire business from one powerful dashboard.

**$60M+ processed · 10M+ transactions · 150,000+ sellers**

---

## Features

### 🏪 Storefronts & Checkout
- Custom storefronts with themeable templates and custom domain support
- Embeddable checkout widget you can drop on any website
- Cart support so customers can purchase multiple products in one order
- Sell file downloads, serial keys, and service/text deliverables

### 💳 60+ Payment Methods, Fees as Low as 0%

| Fiat | Crypto |
|------|--------|
| Stripe | Bitcoin (BTC) |
| PayPal | Ethereum (ETH) |
| Cash App | Litecoin (LTC) |
| Venmo | Solana (SOL) |
| SumUp | Tron (TRX) |
| + more | BNB, USDC, USDT + more |

### 📊 Powerful Dashboard
- Real-time revenue, order, customer, and traffic analytics
- Cashflow charts and performance trends
- Order management with instant delivery tracking
- Audit logs and team permissions

### 🛡️ Built-in Fraud Protection
- VPN and proxy detection
- Disposable email blocking
- CAPTCHA verification
- IP and email blacklisting
- Device fingerprinting

### 🔧 Developer Friendly
- Webhooks for real-time event notifications
- Discord integration for order alerts
- API access for custom integrations
- Embeddable checkout SDK

### 🎟️ Marketing & Growth
- Coupon codes with percentage and fixed discounts
- Customer accounts with order history and balance top-ups
- Feedback and review system
- Affiliate program
- Gift cards

### ⚡ Reliable & Secure
- **99.99% uptime** with infrastructure that never sleeps
- Passwordless login for merchants
- Encrypted file storage
